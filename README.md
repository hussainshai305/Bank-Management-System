# Smart Banking Management System

A Java Swing and MySQL based Banking Management System.

## Features
- Add Account
- View Accounts
- Update Account
- Delete Account
- Deposit Money
- Withdraw Money
- Balance Enquiry
- Transfer Money
- Mini Statement
- Transaction History

## Technologies Used
- Java
- Swing
- JDBC
- MySQL
- Eclipse IDE

## Project Description
This project is a desktop banking application developed using Java Swing for the user interface and MySQL for database management. It provides core banking functionalities such as account management, money transactions, balance enquiry, and transaction tracking.
